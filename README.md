# Excel-Power-Pivot-Revenue-Permit-Analysis-Dashboard
This project demonstrates the use of Excel Power Pivot, DAX, KPIs, PivotTables, PivotCharts, and Slicers to analyze large-scale building permit data and generate meaningful business insights.


The dataset contains over 1.2 lakh+ building permit records, where Power Pivot was used to build a mini data model, create relationships between tables, perform calculations using DAX, and visualize revenue trends across permit categories.

Key Features

Imported and managed large datasets using Power Pivot
Built relationships between multiple tables
Created calculated columns using RELATED()
Developed DAX measures such as:
DISTINCTCOUNT
COUNT
Revenue calculations
Designed KPI indicators for performance tracking
Built interactive PivotTables and PivotCharts
Added Slicers for dynamic filtering
Performed revenue analysis by:
Permit Type
Work Group
Permit Issue Date
